# Proyecto Huella

**Proyecto Huella** es un sistema de control de asistencia diseñado para registrar y gestionar las horas de entrada y salida de los usuarios. Este proyecto fue desarrollado bajo la marca **Security Access**, cuya misión es garantizar la seguridad y protección de los datos de los usuarios, eliminando el uso del papel en favor de soluciones tecnológicas eficientes.

## Descripción del Proyecto

El objetivo principal del **Proyecto Huella** es modernizar los procesos de control de asistencia, permitiendo a los usuarios registrar sus horarios de entrada y salida de manera digital. Con este sistema, se busca no solo optimizar el tiempo, sino también reducir el uso de recursos físicos como el papel, contribuyendo a un entorno más sustentable y eficiente.

## Funcionalidades Principales

- Registro de entrada y salida de usuarios en tiempo real.
- Almacenamiento seguro de la información para garantizar la privacidad de los datos.
- Exportación de los registros a un archivo **Excel**, donde se detallan las horas de ingreso y salida de cada usuario.
- Interfaz intuitiva y fácil de usar.

## Ventajas del Sistema

- **Eliminación del uso de papel**: Ayuda a disminuir el uso de recursos físicos y evita la gestión manual de los registros de asistencia.
- **Automatización de procesos**: Permite llevar un control eficiente y preciso de las asistencias sin intervención humana.
- **Seguridad de los datos**: Gracias a la marca **Security Access**, se garantiza la protección de los datos personales de cada usuario.
